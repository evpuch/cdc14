 # About 
 ## Wrote a paper on genetic engineering for crop pathogens with friends Jeremy Alexander and Jacob Friedman

###Protein Expression & Purification:

- Successfully cloned CpCdc14 gene into pET-15b plasmid 
- Expressed in E. coli BL21 cells with 6x-His tag
- Purified using Ni²⁺-NTA affinity chromatography
- Achieved 11.14 μM protein concentration


###Biochemical Characterization:

- Confirmed phosphatase activity using pNPP assays (99.15% inhibition with sodium tungstate)
- Determined Michaelis-Menten kinetics (KM = 35.59 ± 3.2 mM)
- Mapped substrate specificity to phosphoserine-containing peptides

###Computational Drug Design:

- Built homology model using S. cerevisiae Cdc14 as template 
- Screened 11 potential inhibitors using molecular docking
- Optimized lead compound (aurintricarboxylic acid) improving binding affinity by 3.14 kcal/mol
